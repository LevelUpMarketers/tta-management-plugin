## Ticket Attendees

The ticket editor displays an **Attendees** section for each ticket. This list
shows every attendee who purchased that ticket in a single table. The table
heading contains **Name**, **Email**, **Phone**, **Paid**, **Actions** and
**Refund $** columns. Each attendee occupies one row beneath this heading. The
**Paid** column shows the amount charged for that attendee's ticket. The
**Refund $** field lets admins specify a partial refund before clicking the
**Refund** button. Removing an attendee immediately deletes the attendee
record.
