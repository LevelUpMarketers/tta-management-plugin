## Ticket Attendees

The ticket editor displays an **Attendees** section for each ticket. This list
shows every attendee who purchased that ticket in a single table. The table
heading contains **Name**, **Email**, **Phone** and **Actions** columns. Each
attendee occupies one row beneath this heading. The row provides buttons to
issue a refund or cancel their attendance. Removing an attendee immediately
deletes the attendee record.
